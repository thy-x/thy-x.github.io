Vagrible is a static Vagrant site generator for infrastructure in multiple environments.

It's similar to my older project "Vagrant - Multi-Machine-Multi-Environment" , Vagrible is using Ansible to ensure multiple Vagrantfiles are created simulating multiple environments

It can simulate environments:

  1. Multi OS
  2. Multi Servers
  3. Multi datacenter
  4. Hybrid cloud
